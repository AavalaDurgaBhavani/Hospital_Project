# Healthcare Operations & Patient Risk Intelligence Dashboard

An interactive healthcare analytics dashboard built using Microsoft Power BI to monitor hospital operations, 
patient admissions, financial performance, readmission trends, mortality rates, and resource utilization.
This project demonstrates end-to-end data analytics workflow including data cleaning, data modeling,
DAX calculations, and dashboard design.

# Project Objective

The objective of this project is to transform raw healthcare data into actionable insights that help hospital management:
- monitor patient flow and admissions
- analyze revenue and treatment costs
- identify readmission and mortality trends
- evaluate branch and department performance
- improve operational efficiency and decision-making


# Features

- Interactive KPI Dashboard
- Patient Admission Analysis
- Revenue & Financial Analytics
- Readmission & Mortality Monitoring
- Bed Occupancy Analysis

# Data Cleaning & Transformation

Performed data preprocessing using Power Query:
- standardized inconsistent gender values
- handled null and missing values
- cleaned city and chronic condition columns
- created age groups and chronic flags
- validated admission and discharge data
- calculated length of stay
- transformed billing and payment categories

# Data Model

Implemented a star-schema data model with relationships between:

Patients
Admissions
Billing
Doctors
Departments
Hospital Branches
Treatments
Date Table
